# Workspaces & Windows

Mirador is a multi-window viewer. This is the feature that makes it useful for comparison work,
classroom discussion, and research tasks involving multiple objects.

## Mosaic Workspace

The mosaic workspace arranges windows in a structured layout. It is a good default when readers need
predictable side-by-side comparison.

## Elastic Workspace

The elastic workspace is better for looser exploration. It lets users arrange windows more freely and
compare materials without a fixed grid.

## Window Controls

A window may expose controls to close it, maximize it, choose views, open side panels, move through
canvases, and change thumbnail navigation.

## Comparison Workflow

1. Open two or more manifests.
2. Put related canvases in neighboring windows.
3. Use zoom and pan independently in each window.
4. Keep metadata or attribution panels open only where they help the comparison.
