# Annotations & Search

Mirador can display annotations and IIIF search results when the loaded resource provides the
necessary services.

## Annotation Display

Mirador supports annotation patterns from both IIIF Presentation API v2 and v3. Supported selectors
include fragment selectors and SVG selectors for rectangular and path-based regions.

## Annotation Creation

If the viewer includes annotation authoring tools, use the available annotation controls to create,
edit, or review notes on the canvas.

## Search

When a manifest includes an IIIF Content Search API service, Mirador can expose search in the window
sidebar.
