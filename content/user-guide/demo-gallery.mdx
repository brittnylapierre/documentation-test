## Useful Scenario Demos

- **Empty workspace:** start from a blank viewer and add content.
- **Collections:** load a collection and navigate to manifests.
- **Content search:** test IIIF Content Search API examples.
- **Multiple viewers:** compare pages that show more than one Mirador viewer.
- **Iframe:** try a viewer displayed inside an iframe.
- **Layers:** inspect manifests with layers and image annotation choices.
- **Static images:** use resources without IIIF Image API services.
- **Minimalist viewer:** try a streamlined single-resource display.
- **Right-to-left:** test resources with right-to-left viewing direction.
- **SVG annotations:** inspect path-based annotation regions.
- **Table of contents:** open resources with structural ranges.
- **Audio/video:** try IIIF Presentation API v3 A/V examples.
