## Why Mirador

- It gives researchers a flexible workspace for comparing multiple IIIF resources side by side.
- It supports close visual inspection through zooming, panning, canvas navigation, and multiple view modes.
- It understands IIIF Presentation API v2 and v3 resources, including collections, static images, audio, and video.
- It surfaces the context around an object: manifest metadata, canvas metadata, structures, and tables of contents.
- It can be extended with plugins for annotation creation, image tools, custom menus, and institution-specific workflows.
